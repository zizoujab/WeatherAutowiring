How to install the project
1. get the code
```
git clone git@github.com:zizoujab/WeatherAutowiring.git
```
2. install dependencies
```
cd WeatherAutowiring
composer install
```
3. get the weather
```
php bin/console app:weather 45.2 8.1
```
